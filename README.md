                                ** Bem Vindo ao meu Portifólio Digital ** 

## Passo 1

## Antes de rodar, você precisa instalar as dependências, aperte "crtl + shift + ' " para abrir seu prompt de comando, logo após digite " npm install ".

----------------------------------------------------------------------------------------------------------------------------

## Passo 2

## Depois de baixar as dependências, use o comando " npm run dev " para rodar.

----------------------------------------------------------------------------------------------------------------------------

## Este portifólio contém 9 funcionalidades: 

1- Barra de Progresso de Leitura: Uma barra laranja no topo da tela que cresce horizontalmente conforme o usuário rola a página para baixo, indicando o quanto falta para o fim.

2 - Menu de Navegação Responsivo: Um menu fixo (que recolhe em telas menores) com links que levam suavemente para cada seção (âncoras) e um botão de "Baixar PDF".

3 - Roteamento e Página 404: O app possui um sistema de rotas que detecta se o usuário tentou acessar uma página inexistente e exibe uma tela de "Page not found".

4 - Hero Section com Paralaxe: A seção inicial possui um efeito onde a imagem de fundo se move em uma velocidade diferente do texto durante a rolagem, criando profundidade.

5 - Avatar Interativo (Seção Sobre): Um componente de imagem que alterna entre uma foto "Profissional" e uma "Casual" (Hobby) quando o usuário passa o mouse por cima.

6 - Galeria com Lightbox (Modal): Um grid de imagens onde, ao clicar em uma foto, abre-se uma janela sobreposta (modal) com a imagem ampliada e sua descrição detalhada.

7 - Seletor de Habilidades Dinâmico: Na seção de Skills, há uma lógica interativa onde clicar no ícone de uma habilidade (ex: React, Design) atualiza instantaneamente o texto descritivo e anima a barra de progresso de proficiência ao lado.

8 - Cards de Projetos: Exibição de projetos em cards que possuem efeitos de elevação (hover) e links externos para visualização.

9 - Formulário de Contato Funcional (Mock): Um formulário com campos de e-mail e mensagem. Ao clicar em enviar, ele previne o recarregamento da página e exibe um feedback visual (notificação "Toast") confirmando o envio.
